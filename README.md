# Food Receipt Manager

A simple Angular application for managing food receipts.

This project allows users to create receipts, add multiple items, and view saved receipts.

---

## Features

- Add food receipts
- Add multiple items per receipt
- Remove items dynamically
- View receipt list
- Built using Angular Reactive Forms

## How to Run

Install dependencies
 - npm install

Run Development Server
 - ng serve

Open Browser
 - http://localhost:4200
